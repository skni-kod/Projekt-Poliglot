# Poliglot

### Word-puzzle game based on Polish dictionary.

Use the provided letters to form words and complete the entire puzzle. A level is successfully completed when you guess all the hidden words correctly.

This project utilizes a database of 10000 most popular words in Polish language (source: *Otwarty słownik frekwencyjny leksemów* V.06.2009, author: Jerzy Kazojć.) 

---

### Technologies Used
* **Language:** C++
* **Framework:** Qt 6 (v. >= 6.6.0)
* **IDE:** Qt Creator (v. >= 12.0.0)
* **Compiler:** MinGW

---

### How to run the project on your PC?

**a) Download the release (Recommended!)**

* You can download the pre-compiled version from the [Releases](https://github.com/skni-kod/Projekt-Poliglot/releases/tag/1.0.0) page.

OR

**b) Compile it yourself!**

1. Clone this repository.
2. Configure the project using **Qt Creator** (v. >= 12.0.0).
3. During the configuration choose MinGW toolkit (Qt v. >= 6.6.0).
4. The project should be ready to build and run.
